[Important Flutter CLI Commands]
a. flutter doctor : menampilkan informasi software yang dibutuhkan flutter.
b. flutter create <APP_NAME> : membuat project aplikasi flutter baru di directory tertentu.
c. flutter run <DART_FILE> : menjalankan project aplikasi di device yang tersedia.
d. flutter emulators : menampilkan daftar emulator yang ter-install.
   flutter emulators --launch <EMULATOR_ID> : menampilkan option untuk membuka emulator.
   flutter emulators --create [--name xyz] : menampilkan option untuk membuat emulator baru.
e. flutter channel : menampilkan daftar flutter channel yang tersedia dan menunjukan channel yang digunakan saat ini.
f. flutter pub add <package_name> : menambahkan packages ke dependencies yang ada di pubspec.yaml.
   flutter pub get : men-download semua packages atau dependencies yang ada di pubspec.yaml.
g. flutter build <DIRECTORY> : memproduksi sebuah file aplikasi untuk keperluan deploy atau publish ke AppStore, PlayStore, dll.
h. flutter clean : menghapus folder build serta file lainnya yang dihasilkan saat menjalankan aplikasi di emulator dan perintah ini juga akan memperkecil ukuran project tersebut.
